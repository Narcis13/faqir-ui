Why Faqir wins

Four properties that make it the natural substrate for agent-built interfaces.

Agent-native

Five data attributes are the entire DOM contract. Every component ships a manifest, so an agent can generate markup and machine-verify it against the spec — no screenshots, no guesswork.

Zero lock-in

Plain HTML and CSS. No virtual DOM, no bundler, no framework runtime beyond a tiny optional core. Paste the markup anywhere and it renders — today and in ten years.

Themeable to the bone

Every colour, radius and space is a token. Switch among seven themes across light and dark by swapping one stylesheet — try it live in the header right now.

Accessible by default

WAI-ARIA anatomy baked into every recipe, focus rings from tokens, keyboard support for free, and a WCAG-AAA contrast theme in the box. The audit rules ship with the registry.
