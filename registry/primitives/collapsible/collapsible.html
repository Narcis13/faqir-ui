What is Faqir UI?
Faqir UI is a zero-class, manifest-driven component framework. Components are plain HTML with data attributes — no build step and no runtime required.
How does theming work?
Every color and spacing value is a CSS custom property (a token). Switch themes by swapping a stylesheet — the components themselves never change.
Is JavaScript required?
No. The collapsible is built on the native <details>/<summary> elements — it opens and closes with zero JavaScript, and animates via ::details-content where the browser supports it.
